Afélis, qu'est-ce que c'est ?

Contexte

Animafriends est la suite du projet UX/UI, il s'agit ici de sa landing page réalisé là encore dans le cadre de ma formation de développeur web full-stack. Le but était de réaliser cette application web à l'aide du CMS WordPress.

Fonctionnalités

Animafriends est une application de partage de connaissances sur diverses races de chiens. Les utilisateurs ont la possibilité de créer un journal de bord sur leur chiens ou de consulter ceux d'un autre. S'ils le souhaitent ils ont la possibilité de se contacter entre eux afin de partager plus en détails.
